# 💰 Cotizador de Préstamos

Una aplicación web moderna y funcional desarrollada con **React** y **Vite**, diseñada para calcular cuotas de préstamos en tiempo real con una interfaz intuitiva.

---

## 🚀 Tecnologías Utilizadas

Este proyecto fue construido utilizando las siguientes herramientas:

* **[React.js](https://reactjs.org/):** Biblioteca para construir la interfaz de usuario.
* **[Vite](https://vitejs.dev/):** Herramienta de construcción ultra rápida para el frontend.
* **[Tailwind CSS](https://tailwindcss.com/):** Framework de CSS basado en utilidades para el diseño.
* **[JavaScript (ES6+)](https://developer.mozilla.org/es/docs/Web/JavaScript):** Lógica de programación y manipulación de datos.
* **[Node.js](https://nodejs.org/):** Entorno de ejecución para la gestión de dependencias.
